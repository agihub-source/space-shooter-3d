# 🚀 Space Shooter 3D

一个使用 Three.js 构建的 3D 太空射击游戏！

## 游戏特色

- ✨ **3D 太空场景** - 星空背景 + 粒子效果
- 🎮 **流畅控制** - WASD 移动 + 鼠标瞄准
- 💥 **爆炸特效** - 粒子系统
- 🎯 **分数系统** - 击杀敌人获得积分
- ❤️ **生命系统** - 被撞击扣血

## 控制

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动飞船 |
| 鼠标移动 | 瞄准 |
| 点击 / 空格 | 射击 |
| Q / E | 翻滚 |

## 技术栈

- **Three.js** - 3D 渲染引擎
- **WebGL** - GPU 加速
- **GitHub Pages** - 免费托管

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/agihub-source/space-shooter-3d.git

# 用 HTTP 服务器运行
python -m http.server 8000
# 或
npx serve .
```
